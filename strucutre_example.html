Estructura de una página web.

< HTML >

< body >

< h1 > Este es el encabezado principal < /h1 >

< p >Este puede ser la introducción a la página. y si la página es muy larga se pueden añadir más encabezados.< / p >

< h2 >Este es un ecabezado menos importante.< / h2 >

< p > Los encabezados en HTML (etiquetas < h1 > a < h6 >) son fundamentales para organizar y estructurar el contenido de una página web.
Cada nivel de encabezado tiene un propósito jerárquico, desde < h1 >, que se usa para el título principal de la página, hasta < h6 >, para los subtítulos de menor importancia. < / p >

< h3 >Aquí puedes ver otro encabezado< / h3 >

< / body >

< / HTML >
