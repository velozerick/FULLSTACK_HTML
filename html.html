¿Qué es HTML?

HTML, o HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje básico utilizado para estructurar y presentar contenido en la web. Fue desarrollado por Tim Berners-Lee a finales de los años 80, y se considera el cimiento de cualquier sitio o aplicación web.

Entendiendo la Estructura

HTML usa elementos para describir la estrucutra de las páginas.

La estructura básica de una página HTML comienza con la etiqueta "html", que encierra todo el documento. Dentro de ella, la etiqueta "body" contiene el contenido visible, como encabezados y párrafos.

Los encabezados (h1, h2, h3, etc.) organizan el contenido jerárquicamente: h1 se usa para el título principal, mientras que h2 y h3 dividen el contenido en subsecciones.
Los párrafos (p) agregan texto explicativo debajo de cada encabezado, creando bloques de contenido claros y estructurados para los usuarios y motores de búsqueda.
