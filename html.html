¿Qué es HTML?

HTML, o HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje básico utilizado para estructurar y presentar contenido en la web. Fue desarrollado por Tim Berners-Lee a finales de los años 80, y se considera el cimiento de cualquier sitio o aplicación web.

¿Qué es un IDE?

Un **IDE** (Entorno de Desarrollo Integrado, por sus siglas en inglés) es una aplicación que proporciona herramientas útiles para escribir, editar y depurar código. Un IDE facilita el desarrollo al incluir características como autocompletado, resaltado de sintaxis, y depuración en tiempo real. Existen muchos IDEs, pero uno de los más populares para desarrollo web es **Visual Studio Code (VS Code)**, que es gratuito, personalizable y tiene muchas extensiones que mejoran su funcionalidad.

Instalación de Visual Studio Code

A continuación, te mostramos los pasos para instalar VS Code y preparar tu entorno de desarrollo para crear páginas web. También aprenderás a instalar la extensión **Live Server**, que te permite ver los cambios en tu navegador en tiempo real.

Paso 1: Descarga de Visual Studio Code

Visita la página oficial de Visual Studio Code en https://code.visualstudio.com/ y selecciona el sistema operativo que estás utilizando (Windows, macOS, o Linux). Luego, haz clic en el botón de descarga.

Paso 2: Instalación de Visual Studio Code

Una vez descargado el archivo, ábrelo para iniciar la instalación. Sigue las instrucciones en pantalla para completar el proceso. Asegúrate de marcar la opción de agregar VS Code al PATH si estás en Windows, ya que esto facilitará el uso de la terminal.

Paso 3: Iniciar Visual Studio Code

Abre Visual Studio Code y familiarízate con la interfaz. Verás el panel de archivos a la izquierda, la ventana del editor en el centro y la barra de tareas con herramientas como la terminal integrada en la parte inferior.

Paso 4: Instalar la Extensión Live Server

La extensión **Live Server** permite que tu navegador se actualice automáticamente cada vez que guardas cambios en tu archivo HTML. Para instalarla:

  1. En Visual Studio Code, ve a la barra lateral izquierda y selecciona el ícono de extensiones (parece un cuadrado con cuatro partes).
  2. En la barra de búsqueda, escribe “Live Server”. Busca la extensión de **Ritwick Dey** y haz clic en "Instalar".

Paso 5: Crear tu Primer Proyecto

Ahora que tienes VS Code configurado, puedes empezar tu proyecto de página web:

  1. Crea una carpeta en tu computadora donde guardarás el proyecto (por ejemplo, “MiPaginaWeb”).
  2. Abre esa carpeta en VS Code seleccionando "Archivo" > "Abrir carpeta".
  3. Dentro de la carpeta, crea un archivo llamado `index.html` y abre el archivo en VS Code.
  4. Escribe el siguiente código HTML básico:

            <!DOCTYPE html>
            <html>
                <head>
                    <title>Mi Primera Página</title>
                </head>
                <body>
                    <h1>¡Hola, mundo!</h1>
                    <p>Esta es mi primera página web usando Visual Studio Code.</p>
                </body>
            </html>

Paso 6: Usar Live Server

Para ver tu página en el navegador y que se actualice automáticamente, sigue estos pasos:

  1. Haz clic derecho en el archivo `index.html` en el panel de archivos y selecciona “Open with Live Server”.
  2. Se abrirá tu navegador mostrando la página. Cada vez que guardes cambios en el archivo, la página se actualizará automáticamente.

¡Listo! Ahora estás preparado para desarrollar páginas web en tiempo real usando Visual Studio Code y Live Server.

IDEs en Línea

Los **IDEs en línea** son entornos de desarrollo integrados accesibles desde el navegador. Permiten escribir, ejecutar y probar código sin necesidad de instalar software adicional en tu computadora. Son útiles para trabajar en proyectos rápidos, colaborar con otros desarrolladores y acceder al proyecto desde cualquier dispositivo con conexión a internet.

Ventajas de Usar IDEs en Línea

  • Accesibilidad: Puedes acceder a tu entorno de desarrollo desde cualquier lugar, en cualquier dispositivo.
  • Sin instalación: No necesitas instalar software, lo que ahorra espacio en tu computadora y evita configuraciones complejas.
  • Colaboración: Muchos IDEs en línea permiten la colaboración en tiempo real, facilitando el trabajo en equipo.

Principales Opciones de IDEs en Línea

1. CodeSandbox

**CodeSandbox** es una plataforma popular para desarrollo web que soporta varios lenguajes y frameworks como JavaScript, React, y Node.js. Ofrece un entorno visual completo con soporte para Live Preview, lo que permite ver los cambios en tiempo real.

Pasos para comenzar con CodeSandbox:

  1. Visita https://codesandbox.io.
  2. Regístrate o inicia sesión (puedes usar tu cuenta de GitHub o Google).
  3. Selecciona “Create Sandbox” y elige la plantilla HTML o el framework de tu preferencia.
  4. Empieza a escribir tu código en el editor. La vista previa se actualizará automáticamente.

2. Project IDX

**Project IDX** es un IDE en línea desarrollado por Google, diseñado para crear aplicaciones web y móviles. IDX permite desarrollo en múltiples lenguajes y frameworks como JavaScript, Python, y Flutter. Es ideal para proyectos de desarrollo web y móvil con características avanzadas de integración y colaboración.

Pasos para comenzar con Project IDX:

  1. Visita https://projectidx.dev (requiere cuenta de Google para iniciar).
  2. Inicia un nuevo proyecto seleccionando el tipo de aplicación que deseas crear (web o móvil).
  3. Accede a las herramientas integradas de desarrollo, como el editor de código, terminal y vista previa en tiempo real.

¿Cuál Debo Elegir?

Ambos IDEs en línea son excelentes opciones, y la elección depende de tus necesidades:

  • **CodeSandbox** es ideal para proyectos de desarrollo web rápido, especialmente si trabajas con JavaScript o frameworks como React.
  • **Project IDX** es una opción avanzada para aquellos que desean crear aplicaciones web y móviles con herramientas de Google y soporte para varios lenguajes.

Ambos entornos te permiten empezar a programar de inmediato, sin configuraciones complicadas, y ofrecen una experiencia similar a la de un IDE de escritorio.

Entendiendo la Estructura

HTML usa elementos para describir la estrucutra de las páginas.

La estructura básica de una página HTML comienza con la etiqueta "html", que encierra todo el documento. Dentro de ella, la etiqueta "body" contiene el contenido visible, como encabezados y párrafos.

Los encabezados (h1, h2, h3, etc.) organizan el contenido jerárquicamente: h1 se usa para el título principal, mientras que h2 y h3 dividen el contenido en subsecciones.
Los párrafos (p) agregan texto explicativo debajo de cada encabezado, creando bloques de contenido claros y estructurados para los usuarios y motores de búsqueda.

Estructura Básica de un Documento HTML

Un documento HTML tiene una estructura específica que permite que el navegador interprete y muestre correctamente el contenido. A continuación, te mostramos la estructura básica y qué significa cada elemento:


            <!DOCTYPE html>
            <html>
                <head>
                    <meta charset="UTF-8">
                    <title>Título de la Página</title>
                </head>
                <body>
                   
                </body>
            </html>

Explicación de Cada Parte

  • <!DOCTYPE html>: Indica al navegador que el documento está escrito en HTML5. Este es un estándar para asegurar que el documento se renderice de manera correcta y moderna.
  • <html>: La etiqueta <html> es el contenedor principal de todo el documento HTML. Todo el contenido visible e invisible de la página debe estar contenido dentro de esta etiqueta.
  • <head>: El elemento <head> contiene metadatos sobre el documento. Aquí se incluyen información que no es visible en la página, como el título, enlaces a hojas de estilo, y configuraciones de codificación.
  • <meta charset="UTF-8">: Define la codificación de caracteres como UTF-8, que es compatible con la mayoría de los idiomas y caracteres. Esto asegura que caracteres especiales se muestren correctamente en la página.
  • <title>: El contenido dentro de la etiqueta <title> se muestra en la pestaña del navegador cuando se carga la página. También es importante para el SEO (optimización de motores de búsqueda) ya que describe el contenido de la página.
  • <body>: El elemento <body> contiene todo el contenido visible de la página web. Aquí se incluyen elementos como encabezados, párrafos, imágenes, enlaces, etc.

Esta estructura es el punto de partida para cualquier documento HTML y define los elementos esenciales que todos los desarrolladores web deben conocer.

Tags

Los tags, o etiquetas, son los elementos básicos en HTML que le indican al navegador cómo debe estructurarse y presentarse el contenido de una página web. Cada tag representa un elemento o tipo de contenido específico, como un encabezado, un párrafo, una imagen, o un enlace.


Las etiquetas HTML están encerradas entre signos de menor y mayor (`< >`). La mayoría de las etiquetas tienen una etiqueta de apertura, que indica el inicio de un elemento, y una etiqueta de cierre, que señala su final. Por ejemplo:

<p>Este es un párrafo en HTML</p>

En este caso, la etiqueta de apertura es `< p >` y la de cierre es `< / p >` Todo el contenido entre estas etiquetas se considera un elemento de párrafo y será presentado como tal en la página web.



A continuación, se muestra una tabla con algunos de los tags más utilizados en HTML y su propósito:

<html> Documento HTML Marca el inicio y el final de un documento HTML.
<head> Encabezado del Documento Contiene metadatos, como el título, enlaces a estilos y scripts.
<title> Título de la Página Define el título que aparece en la pestaña del navegador.
<body> Cuerpo del Documento Contiene el contenido visible de la página, como texto, imágenes y enlaces.
<h1> a <h6> Encabezados Define títulos y subtítulos. H1 es el de mayor importancia y H6 el de menor.
<p> Párrafo Crea un bloque de texto para contenido descriptivo.
<a> Enlace Crea un enlace a otra página o recurso con el atributo href.
<img> Imagen Incorpora una imagen en la página. Requiere el atributo src.
<ul> y <ol> Listas Crea listas. UL es para listas con viñetas y OL para listas numeradas.

<div> División de Contenido Agrupa contenido para aplicar estilos o scripts específicos.

Estos tags son algunos de los más comunes en HTML y ayudan a definir la estructura y funcionalidad de una página web.

Elementos

En HTML, un **elemento** es una combinación de una etiqueta de apertura, el contenido y, en la mayoría de los casos, una etiqueta de cierre. Los elementos son las piezas fundamentales que definen el contenido y la estructura de una página web. Cada elemento tiene un propósito específico, como organizar el texto, insertar imágenes, crear enlaces, entre otros.

A continuación, se presenta una tabla con algunos de los elementos HTML más utilizados y su propósito:

Elemento Descripción Ejemplo

<a> Enlace <a href="https://example.com">Visita nuestro sitio</a>

<ul> / <ol> Listas <ul><li>Elemento 1</li><li>Elemento 2</li></ul>

Estos elementos son algunos de los más comunes en HTML y son esenciales para estructurar, organizar y darle funcionalidad a una página web.

Diferencia entre Tags y Elementos

En HTML, la diferencia entre tags y elementos es sutil pero importante. Un **tag** (etiqueta) es la parte del código que define el inicio y, en la mayoría de los casos, el fin de un contenido específico. Por ejemplo, en el código `<p>Texto</p>`, `<p>` es un tag de apertura y `</p>` es un tag de cierre.

Un **elemento**, en cambio, incluye el tag de apertura, el contenido y el tag de cierre. En el ejemplo anterior, `<p>Texto</p>` en su totalidad es un elemento de párrafo. En otras palabras, el elemento es el conjunto de los tags y su contenido.

Para resumir: los tags son las etiquetas individuales que definen el tipo de contenido, mientras que los elementos son las estructuras completas que incluyen los tags y el contenido en sí.

Encabezados

Los encabezados en HTML, desde <h1> hasta <h6>, se utilizan para estructurar y organizar el contenido. Cada encabezado tiene un nivel de importancia, donde <h1> es el título principal de la página y <h6> el de menor relevancia. Los encabezados permiten que los usuarios y motores de búsqueda entiendan la jerarquía del contenido en la página. Por ejemplo:

<h1>Título Principal</h1>
            <h2>Subtítulo Nivel 1</h2>
            <h3>Subtítulo Nivel 2</h3>

En una página HTML, es buena práctica usar solo un <h1> para el título principal y luego <h2> y otros encabezados para dividir el contenido en subsecciones.

Párrafos

El elemento <p> se usa para definir bloques de texto o párrafos en HTML. Cada vez que se necesita una nueva sección de texto, el contenido se coloca dentro de una etiqueta <p>. Esto facilita la lectura y estructura el contenido en fragmentos. Un ejemplo de uso del elemento de párrafo sería:

<p>Este es un párrafo de texto.</p>
            <p>Este es otro párrafo con información adicional.</p>

Negrita, Cursiva y Énfasis

Las etiquetas <b>, <i> y <em> se utilizan para dar formato al texto, haciéndolo más legible y dándole un mayor énfasis en ciertos casos. Aquí te explico cada una:

  • <b> (Negrita): Se usa para destacar texto sin darle un significado adicional. Ejemplo: <b>Texto en negrita</b>
  • <i> (Cursiva): Se usa para poner el texto en cursiva. Puede indicar algo de menor importancia o una cita. Ejemplo: <i>Texto en cursiva</i>
  • <em> (Énfasis): Es similar a la cursiva, pero con la intención de darle un mayor significado al texto, útil para accesibilidad. Ejemplo: <em>Texto con énfasis</em>

Estructura y Marcado Semántico

El marcado semántico en HTML utiliza etiquetas que describen el propósito del contenido. Este tipo de marcado mejora la accesibilidad y ayuda a los motores de búsqueda a entender el contenido de la página. Ejemplos de etiquetas semánticas son:

  • <header>: Define la cabecera de la página o sección, que puede incluir logotipos, menús o títulos.
  • <nav>: Contiene enlaces de navegación para el sitio.
  • <main>: Contiene el contenido principal de la página.
  • <section>: Agrupa contenido relacionado en una sección.
  • <article>: Representa contenido independiente, como un artículo o publicación.
  • <footer>: Define el pie de página de la página o sección, que puede incluir derechos de autor o información de contacto.

El uso adecuado de estos elementos mejora la estructura y hace que el código sea más claro y fácil de leer para los usuarios y los motores de búsqueda.

Practicando un Código Básico

Escribe el siguiente código HTML en el cuadro de texto de abajo y luego presiona el botón "Verificar Código" para comprobar si está correcto:


            <!DOCTYPE html>
            <html>
                <head>
                    <title>Página de Práctica</title>
                </head>
                <body>
                    <h1>Bienvenido a HTML</h1>
                    <p>Este es un párrafo introductorio donde puedes escribir sobre el propósito de la página.</p>
                    <h2>Sección de Contenido</h2>
                    <p>Aquí puedes agregar más información o detalles sobre el tema de la página.</p>
                    <a href="https://example.com">Visita nuestro sitio web</a>
                </body>
            </html>

Listas Numeradas

Para crear una lista numerada en HTML, se usa el elemento <ol> (ordered list). Este elemento genera una lista ordenada donde cada ítem tiene un número de orden. Cada elemento de la lista se coloca dentro de una etiqueta <li> (list item).


            <ol>
                <li>Primer elemento</li>
                <li>Segundo elemento</li>
                <li>Tercer elemento</li>
            </ol>

El resultado se verá así:

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Listas con Viñetas

Las listas con viñetas se crean con el elemento <ul> (unordered list). Este tipo de lista no tiene un orden específico y muestra cada elemento con una viñeta. Cada ítem de la lista también se define con la etiqueta <li>.


            <ul>
                <li>Elemento uno</li>
                <li>Elemento dos</li>
                <li>Elemento tres</li>
            </ul>

El resultado se verá así:

  • Elemento uno
  • Elemento dos
  • Elemento tres

Listas de Definición

Las listas de definición se crean usando el elemento <dl> (definition list). Este tipo de lista se usa para mostrar términos y sus respectivas definiciones. Dentro de <dl>, los términos se definen con <dt> (definition term) y las definiciones con <dd> (definition description).


            <dl>
                <dt>HTML</dt>
                <dd>Lenguaje de marcado para crear páginas web.</dd>
            
                <dt>CSS</dt>
                <dd>Lenguaje de estilos para diseñar páginas web.</dd>
            </dl>

El resultado se verá así:

HTML
Lenguaje de marcado para crear páginas web.
CSS
Lenguaje de estilos para diseñar páginas web.

Imágenes en HTML

En HTML, puedes insertar imágenes usando la etiqueta <img>. Esta etiqueta es auto-contenida y no requiere una etiqueta de cierre. El atributo src especifica la ruta de la imagen, mientras que alt proporciona una descripción de la imagen para accesibilidad.


            <img src="ruta/de/imagen.jpg" alt="Descripción de la imagen">

Tablas en HTML

Las tablas en HTML se crean con la etiqueta <table> y están organizadas en filas (<tr>) y celdas (<td>). La etiqueta <th> se utiliza para definir encabezados de columna.


            <table>

                    <th>Encabezado 1</th>
                    <th>Encabezado 2</th>
                </tr>
                <tr>
                    <td>Celda 1</td>
                    <td>Celda 2</td>
                </tr>
            </table>
