¿Qué es HTML?

HTML, o HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje básico utilizado para estructurar y presentar contenido en la web. Fue desarrollado por Tim Berners-Lee a finales de los años 80, y se considera el cimiento de cualquier sitio o aplicación web.

Entendiendo la Estructura

HTML usa elementos para describir la estrucutra de las páginas.

La estructura básica de una página HTML comienza con la etiqueta "html", que encierra todo el documento. Dentro de ella, la etiqueta "body" contiene el contenido visible, como encabezados y párrafos.

Los encabezados (h1, h2, h3, etc.) organizan el contenido jerárquicamente: h1 se usa para el título principal, mientras que h2 y h3 dividen el contenido en subsecciones.
Los párrafos (p) agregan texto explicativo debajo de cada encabezado, creando bloques de contenido claros y estructurados para los usuarios y motores de búsqueda.

Tags

Los tags, o etiquetas, son los elementos básicos en HTML que le indican al navegador cómo debe estructurarse y presentarse el contenido de una página web. Cada tag representa un elemento o tipo de contenido específico, como un encabezado, un párrafo, una imagen, o un enlace.


Las etiquetas HTML están encerradas entre signos de menor y mayor (`< >`). La mayoría de las etiquetas tienen una etiqueta de apertura, que indica el inicio de un elemento, y una etiqueta de cierre, que señala su final. Por ejemplo:

<p>Este es un párrafo en HTML</p>

En este caso, la etiqueta de apertura es `< p >` y la de cierre es `< / p >` Todo el contenido entre estas etiquetas se considera un elemento de párrafo y será presentado como tal en la página web.



A continuación, se muestra una tabla con algunos de los tags más utilizados en HTML y su propósito:

<body> Cuerpo del Documento Contiene el contenido visible de la página, como texto, imágenes y enlaces.
<h1> a <h6> Encabezados Define títulos y subtítulos. H1 es el de mayor importancia y H6 el de menor.
<p> Párrafo Crea un bloque de texto para contenido descriptivo.

<img> Imagen Incorpora una imagen en la página. Requiere el atributo src.

<div> División de Contenido Agrupa contenido para aplicar estilos o scripts específicos.

Estos tags son algunos de los más comunes en HTML y ayudan a definir la estructura y funcionalidad de una página web.

Elementos

En HTML, un **elemento** es una combinación de una etiqueta de apertura, el contenido y, en la mayoría de los casos, una etiqueta de cierre. Los elementos son las piezas fundamentales que definen el contenido y la estructura de una página web. Cada elemento tiene un propósito específico, como organizar el texto, insertar imágenes, crear enlaces, entre otros.

A continuación, se presenta una tabla con algunos de los elementos HTML más utilizados y su propósito:

Estos elementos son algunos de los más comunes en HTML y son esenciales para estructurar, organizar y darle funcionalidad a una página web.

Diferencia entre Tags y Elementos

En HTML, la diferencia entre tags y elementos es sutil pero importante. Un **tag** (etiqueta) es la parte del código que define el inicio y, en la mayoría de los casos, el fin de un contenido específico. Por ejemplo, en el código `<p>Texto</p>`, `<p>` es un tag de apertura y `</p>` es un tag de cierre.

Un **elemento**, en cambio, incluye el tag de apertura, el contenido y el tag de cierre. En el ejemplo anterior, `<p>Texto</p>` en su totalidad es un elemento de párrafo. En otras palabras, el elemento es el conjunto de los tags y su contenido.

Para resumir: los tags son las etiquetas individuales que definen el tipo de contenido, mientras que los elementos son las estructuras completas que incluyen los tags y el contenido en sí.

Encabezados

Los encabezados en HTML, desde <h1> hasta <h6>, se utilizan para estructurar y organizar el contenido. Cada encabezado tiene un nivel de importancia, donde <h1> es el título principal de la página y <h6> el de menor relevancia. Los encabezados permiten que los usuarios y motores de búsqueda entiendan la jerarquía del contenido en la página. Por ejemplo:

<h1>Título Principal</h1>
            <h2>Subtítulo Nivel 1</h2>
            <h3>Subtítulo Nivel 2</h3>

En una página HTML, es buena práctica usar solo un <h1> para el título principal y luego <h2> y otros encabezados para dividir el contenido en subsecciones.

Párrafos

El elemento <p> se usa para definir bloques de texto o párrafos en HTML. Cada vez que se necesita una nueva sección de texto, el contenido se coloca dentro de una etiqueta <p>. Esto facilita la lectura y estructura el contenido en fragmentos. Un ejemplo de uso del elemento de párrafo sería:

<p>Este es un párrafo de texto.</p>
            <p>Este es otro párrafo con información adicional.</p>

Negrita, Cursiva y Énfasis

Las etiquetas <b>, <i> y <em> se utilizan para dar formato al texto, haciéndolo más legible y dándole un mayor énfasis en ciertos casos. Aquí te explico cada una:

  • <b> (Negrita): Se usa para destacar texto sin darle un significado adicional. Ejemplo: <b>Texto en negrita</b>
  • <i> (Cursiva): Se usa para poner el texto en cursiva. Puede indicar algo de menor importancia o una cita. Ejemplo: <i>Texto en cursiva</i>
  • <em> (Énfasis): Es similar a la cursiva, pero con la intención de darle un mayor significado al texto, útil para accesibilidad. Ejemplo: <em>Texto con énfasis</em>

Estructura y Marcado Semántico

El marcado semántico en HTML utiliza etiquetas que describen el propósito del contenido. Este tipo de marcado mejora la accesibilidad y ayuda a los motores de búsqueda a entender el contenido de la página. Ejemplos de etiquetas semánticas son:

  • <header>: Define la cabecera de la página o sección, que puede incluir logotipos, menús o títulos.
  • <nav>: Contiene enlaces de navegación para el sitio.
  • <main>: Contiene el contenido principal de la página.
  • <section>: Agrupa contenido relacionado en una sección.
  • <article>: Representa contenido independiente, como un artículo o publicación.
  • <footer>: Define el pie de página de la página o sección, que puede incluir derechos de autor o información de contacto.

El uso adecuado de estos elementos mejora la estructura y hace que el código sea más claro y fácil de leer para los usuarios y los motores de búsqueda.

Practicando un Código Básico

Escribe el siguiente código HTML en el cuadro de texto de abajo y luego presiona el botón "Verificar Código" para comprobar si está correcto:


            <!DOCTYPE html>
            <html>
                <head>
                    <title>Página de Práctica</title>
                </head>
                <body>
                    <h1>Bienvenido a HTML</h1>
                    <p>Este es un párrafo introductorio donde puedes escribir sobre el propósito de la página.</p>
                    <h2>Sección de Contenido</h2>
                    <p>Aquí puedes agregar más información o detalles sobre el tema de la página.</p>
                    <a href="https://example.com">Visita nuestro sitio web</a>
                </body>
            </html>
